AngularJS Foundation Training

Learning the Superheroic JavaScript MVW Framework

Welcome to the AngularJS Training. On this site you will find all information needed to successfully follow the course. Before coming to the course, you should make sure you have followed the courses as listed under "Required knowledge". Also, you should prepare your laptop by following the steps under "Laptop preparations".

Required knowledge

To follow this course, you will need to have some basic knowledge of web development. Before attending, we advise you to follow the following courses:

  • HTML/CSS basics @ Code Academy
  • JavaScript basics @ Code Academy

Laptop preparations

Make sure your laptop is up to date with the latest versions of the following software:

  • Install Git
  • Install NodeJS
  • Install an IDE like WebStorm or Sublime Text

There are some issues running git from within the Capgemini network. Run the following command to fix this:

$ git config --global url."https://".insteadOf git://

Angular app from Scratch: Getting Started
