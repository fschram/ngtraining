Welcome

Welcome to the AngularJS Foundation Training. On this site you will find all information needed to successfully follow the course. Before coming to the course, you should make sure you have followed the courses as listed under Required knowledge. Also, you should prepare your laptop by following the steps under Laptop preparations.

Required knowledge

To follow this course, you will need to have some basic knowledge of web development. Before attending, we advise you to follow the following courses:

HTML

Start the HTML course at Codecademy.

CSS

Start the CSS course at Codecademy.

JavaScript

Start the JavaScript Fundamentals course at Codecademy.

Laptop preparations

Make sure your laptop is up to date with the latest versions of the following software:

Git

Install the distributed version control system Git.

NodeJS

Install the JavaScript platform NodeJS.

IDE

Install an IDE like Brackets, WebStorm or Sublime Text.

Starting the course

The training is based on a set of AngularJS training videos on Egghead.io called AngularJS App From Scratch: Getting Started. Thre course includes a bootstrapping, data binding, controllers, filters and more.

To start developing you will need to download the starting code from Github. There Git protocol is not supported from within our network. Run the following command for a workaround: $ git config --global url."https://".insteadOf git://

And do not forget to visit the AngularJS website.

Extra

Build

Build your application with the help of frameworks like Bower, Grunt and Gulp.

Material Design
